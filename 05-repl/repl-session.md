#### 'remainder' or modulo operator

```python
>>> 1 % 3
1
>>> -1 % 3
2
>>> 3 % 3
0
>>> 21 % 3
0
```

#### fizz

```python
>>> 3 % 3 == 0
True
>>> 4 % 3 == 0
False
```

#### fizzbuzz

```python
>>> 15 % 3 == 0 and 15 % 5 == 0
True
>>> 3 % 3 == 0 and 3 % 5 == 0
False
```
